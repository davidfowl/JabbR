Features

Web based client

  • Responsive and mobile friendly
  • Works on all modern browsers

Easy Setup

  • Run self hosted, on Windows Azure or AppHarbor

Flexible Authentication

  • Custom username/password
  • Windows Authentication
  • Social Authentication (Facebook, Google, Twitter etc.)

Public and private chat rooms

  • Create an unlimited number of rooms
  • Control access to private rooms using ACLs or invite codes
  • Linkable rooms allows you to send anybody a url to go directly to a room

Emoticons

  • We support Emoji!

File uploads

Shortcut keys

Powerful IRC style commands

Profiles

  • Assign a gravatar to your nick. Be recognized, even in JabbR!
  • Pick a flag to represent your country

Notifications

  • Integrated into Chrome to provide you with popup desktop notifications.
  • Audio notifications.

Extensible

  • Display inline content
